# Portfolio (KSDRILL)

Personal engineering portfolio — Next.js 14, config-driven content, flagship case studies, tech stack, and contact.

## Live site

**https://ksdrill-portfolio.vercel.app**

Share that link anywhere (LinkedIn, résumé, email). It is the public production URL on Vercel.

## Repository

**https://github.com/KSDRILL/ksdrill-portfolio**

## Stack

- Next.js 14 (App Router), React 18, TypeScript
- Tailwind CSS
- TanStack Query, Zod, Lucide icons
- Content: JSON under `config/` (theme, profile, flagship systems, tech stack, navigation, etc.)

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build
```

## Deployment

Production deploys on **Vercel** from this repository when Git is connected, or via `npx vercel --prod` from the project root.
